# 📚 스케줄 관리 프로그램
이 프로젝트는 일정을 관리하는 프로젝트로 일상에서 일정을 관리할 때 도움이 되고자 프로젝트를 진행하게 되었다

이 프로그램은 스케줄을 잘 수행하고 알람과 타이머를 설정하여 집중하는 데에 도움이 될 수 있게 계획 했고 또한 알람과 타이머를 특정 시간 이상 설정 한 후 그 시간이 지나게 되면 포인트를 지급하여 상점에서 아바타를 꾸밀 수 있는 기능 또한 포함하여 수집욕을 자극시켜 집중을 하는 것에 도움을 주었다

프론트 화면은 LLM을 사용하여 디자인 및 개발을 진행 하였고 백엔드 개발하는 곳에 어려움이 있을 때 LLM을 사용하여 개발을 진행하였다

# 🛠️ 주요기능 🛠️
- 알람 설정, 취소 기능 및 알람의 음악을 설정하는 기능과 알람을 설정한 시간이 되면 울리는 기능
- 타이머 설정, 취소 기능 및 타이머가 00 : 00이 되면 타이머가 울리는 기능
- 스케줄 설정, 취소 기능 및 스케줄 수정과 스케줄의 색 설정 기능
- 상점에서 의상, 모자 구매 기능
- 인벤토리에서 상점에서 구매한 의상, 모자 착용 기능
- 본인이 설정한 알람의 기록을 확인하는 기능
- 로그인, 회원가입, 로그아웃, 회원탈퇴, 이메일 인증 기능

# ⭐ 기술 스택 ⭐
- BackEnd : Spring Boot, Spring Boot Security, Thymeleaf
- FrontEnd : HTML, CSS, JavaScript
- DataBase : MySQL

# 🛡️ 보완할 점 🛡️
- 현: 전채적인 속도가 느림
  - 후: 성능을 최적화
- 현: 상점의 의상, 모자등의 아이템 가격과 아이템의 효능등이 존재하지 않음
  - 후: 의상, 모자등의 아이템 추가 및 세트 효과 같은 아이템의 효과를 추가 예정
